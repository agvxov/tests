# fu-shell
GIMP defines the Lisp dialect named "fu-script" as its official scripting language.
Further more, GIMP processes act like servers and can eval fu-script requests.

Now, I was eager to write a *shell* (REPL CLI client) for it when I learned that there was none.
Then, I had to realize that there is no documentation for fu-script what so ever.
I *think* fu-script is also being slowly deprecated in favour of Python.
