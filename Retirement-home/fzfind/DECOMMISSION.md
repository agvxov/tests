specifically to have some sort of file completion.

therefor I never used it and since I never used it,
I never felt the need to fix it.

Since then, I have given up on Bash and the sentiment
that just because the POSIX shell is default,
it must be *good*.

As a result, I will never fix it now, or use it,
or recommend anyone to use it.
